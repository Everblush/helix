# Installation 

## Linux

- Get the .toml configuration file 
```sh 
cd $HOME/.config/helix
wget https://raw.githubusercontent.com/Everblush/helix/everblush.toml
```
- Add the following lines to your `$HOME/.config/helix` 
```sh 
theme = "everblush"
```
